# Freddy's Casting Mod

A spectator camera mod for **Gorilla Tag** and **Project Kitty**, built for content creators and streamers. Features a smooth spectator camera, player nametags, WASD fly mode, and more.

## Features

- Spectator camera with multiple smoothing modes (Lerp, SmoothDamp, Expo, Lunar)
- Player nametags with customizable font, size, and position
- WASD fly mode with stationary option and adjustable speed
- Photon room tools — join by code, disconnect, AFK kick prevention
- Quick target switching (press 1–9 to snap to any player)
- All settings auto-save via BepInEx config

## Installation

1. Install [BepInEx 5.x](https://github.com/BepInEx/BepInEx/releases) into your game folder
2. Download the `.dll` for your game from the website
3. Drop it into `BepInEx/plugins/`
4. Launch the game and press `TAB` to open the menu

## Downloads

Visit the website to download the mod for your game:

- **Gorilla Tag** — `FCM_GorillaTag.dll`
- **Project Kitty** — `FCM_ProjectKitty.dll`
